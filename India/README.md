# India - Financial Model Outputs
This folder contains visual outputs from the India-specific portion of the cross-market model. Each chart is directly extracted from the Master model and represents:

# Included files

1. RAPI
Shows sector-level risk-adjusted return ranking for India

2. Heatmap (Fundamental Pillar scorecard)
Displays sector performance across 5 pillars:
> Profitability
> Efficiency
> Leverage
> Growth
> Valuation

3. CAS (Composite Attractiveness Score)
India's strategic allocation weighting based on fundamentals + macro + risk.

# Purpose
These visuals help illustrate:

> Structural strengths of Indian Sectors
> Cross-sector divergence within India
> Long-term risk-return profile
> Allocation signals generated by CAS & RAPI
