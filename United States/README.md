# United States — Financial Model Outputs (2020–2030)
This folder contains U.S.-specific visual outputs extracted from the cross-market financial model.
These charts provide a complete overview of U.S. sector fundamentals and allocation signals.

# Included Files

1️. RAPI (Risk-Adjusted Performance Index)
Ranks U.S. sectors by long-term risk-adjusted performance.

2️. Heatmap (Fundamental Pillar Scorecard)

Scores sectors across:

> Profitability
> Efficiency
> Leverage
> Growth
> Valuation

3️. CAS (Composite Allocation Score)
Shows U.S. sector allocation weightings for 2025–2030.

# Purpose
These charts help interpret:

✔ Sector-level strengths & weaknesses
✔ Efficiency vs valuation mismatches
✔ Long-term allocation advantages
✔ Cross-market comparison with India
